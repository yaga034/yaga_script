## CameraShake_Aescript

After Effectsで簡単にカメラ手振れエフェクトを実現するスクリプトです。

### 機能紹介

カメラの手振れエフェクトを手軽に適用できます。詳しい使い方は解説動画をご覧ください。

### 解説動画

詳しい使い方は[YouTube](https://www.youtube.com/watch?v=MH9w03IbnvU)でご確認いただけます。
